svgType

Turns plain svg files into printable type for use in 3D printers

Only SVG files are supported. Currently, only paths are processed. Some lines, circles, rectangles are ignored if not inside a path. Text is also ignored. Need some inspiration, check out thenounproject.com. Or create your own using inkscape.

Scaling and height of the Final 3D block depends on the length and width of the SVG. Some are small (100px) while others can be very large (1000px or greater). SVGs can also have different units. Only pixels are used. They are converted into millimeters later on. Scale the length and width acordingly in order to produce properly (height) proportioned type.

You can also adjust the X and Y padding between the SVG and the edge of the block. This is given in a percentage of the block length and width.

Neck Height is also set here. This is typically set to 1 or 2mm. It is automatically scaled when the Reize button is clicked.

Use the resize button to get started. Use sliders to adjust settings. Once satisifed, continue on to the next step.
